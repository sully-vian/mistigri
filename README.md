# Mistigri

Mistigri is a Chat Application made using [Next.js](https://nextjs.org/) with the [Mistral](https://mistral.ai/) public API.

This project was made in the context of an [internship application](https://jobs.lever.co/mistral/a6980b07-e55a-427c-a985-38ecd1e2eea6).

Special thanks to [@sakofchit](https://github.com/sakofchit) for the [System.css](https://sakofchit.github.io/system.css/) library and its unique look.
